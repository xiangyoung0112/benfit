🌐 代理 Proxy 設定（現價查詢）

瀏覽器直接連線台灣證交所 API 時會被 CORS 封鎖。選擇一個代理方式：
方案1（推薦）：填入你的 GAS URL（上方），系統會透過 GAS 抓取現價。
方案2：使用公開 CORS Proxy（速度較慢，有流量限制）。
方案3：自訂 Proxy URL，格式：{url} 會被替換為目標網址。
